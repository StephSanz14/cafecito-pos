## Instalación del proyecto en Frontend

Nos ubicamos en la carpeta general de Frontend y ejecutamos el comando

```bash
ng new cafecito-frontend --directory . --routing --style css --standalone --ssr false
```
Eso nos crea el proyecto sobre la carpeta existente y nos genera nuestro app component

Para correr FrontEnd lo hacemos desde la carpeta "frontend" y ejecutamos el sig comando

```bash
ng serve
```
