# Cafecito Feliz

## Project Overview
Cafecito Feliz is a full-stack application that allows users to manage coffee orders and sales. The project consists of a backend built with Node.js and Express, and a frontend developed using Angular.

## Features
- User authentication and authorization
- Product management
- Order processing
- Admin and seller roles

## Technologies Used
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Frontend:** Angular, TypeScript, Tailwind CSS

## Getting Started

### Prerequisites
- Node.js and npm installed
- MongoDB installed and running

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd cafesito-pos
   ```
2. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```
3. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```

### Running the Application
1. Start the backend server:
   ```bash
   cd backend
   npm start
   ```
2. Start the frontend application:
   ```bash
   cd frontend
   ng serve
   ```
